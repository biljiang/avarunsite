SP500 Forecast by RNN

The Chart below shows SP500 index forecast by a simple RNN model, predicting index values for next five trading days at 9:30 PM (US/Eastern time) each day according the closing index values of that day and preceding 29 days.  （Blue line is actual index values. Orange line is predicted values after last trading days closing. Green line is forecast values for next five trading days.)

Next table shows the forecast values for next five days and their probabilities -the most likely instances are adopted in the chart above. The model will generate a probability distribution for the index to be forecast and choose the most likely instances as its forecast values. The top 2 forecast values and their probabilities would be helpful.

Note:

  • The forecast value is actually a range around the number instead of a precise value. And the probability is the probability of the value falling into the range. Giving out a precise value is for simplicity reason and easiness to demo in chart!
  • The probability is a result of the model's algorithm, not the probability of actual occurrence. It is more meaningful when compared to itself than considering the absolute value.
  • Forecast value for next day is better than the rest, if any good.
